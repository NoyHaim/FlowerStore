Flowers Boutique

Hi, My name is Noy Haim.

I am selling flowers.

The additional pages I created:

1. Wishlist - users can add add items to a "wish list" which are items you want but not yet want to buy.
- Go to /private/wishlist.html

2. Purchases - users can see their previous orders.
- Go to /private/purchases.html

3. Home page.
- Go to /private/home.html

What was hard for me?
Getting use to work with redis and the design of the pages.

I did the project alone.

Supported routes:
- /login
- /register
- /private/home
- /private/store
- /private/add-to-cart
- /private/remove-from-cart
- /private/add-to-wishlist
- /private/remove-from-wishlist
- /private/cart
- /private/wishlist
- /private/purchases
- /private/checkout
- /private/admin (for admin only)
- /private/logout

I made my store secured by adding try/catch to the routs
which helps me recognize unauthorized accesses to the website routs.
I also verified for every user who asks for access to a private route,
using cookies and an "active users" dictionary which stores the users that preformed a valid login.

I did not use react.
